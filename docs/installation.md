# Installation

## Cargo

Install from [crates.io](https://crates.io/crates/databow):

```sh
cargo install databow --version 0.1.0-alpha
```

## Source

Install from source:

```sh
git clone https://github.com/columnar-tech/databow.git
cargo install --path .
```
